Credimap is a web platform that helps students discover, review, and evaluate nonprofits & youth organizations based on real experiences. The goal is to increase transparency, credibility, and accessibility when choosing programs, opportunities, or organizations to engage with.

The platform combines organization profiles, user reviews, ratings, and map-based discovery into a single, intuitive interface.

Features:

Organization Discovery
- Browse nonprofits & youth organizations
- Search and filter by category (e.g. STEM, CS, Psychology), location, and format (e.g. In-person, Hybrid, Remote)
- View organizations on an interactive map

Reviews & Ratings
- Star-based rating system
- Written reviews from students and participants
- Automatic calculation of average ratings

Organization Profiles
- Public profile pages with:
- Description and mission
- Category, location and format
- Reviews and ratings
- Social links (website, LinkedIn, Instagram, Tiktok)

Tech Stack:

Frontend
- React
- Chakra UI
- Axios
- react-simple-maps

Backend
- Node.js
- Express (v4)
- MongoDB
